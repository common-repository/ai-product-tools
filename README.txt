== Installation ==
 
1. Upload 'ai-product-tools.zip' to your '/wp-content/plugins/' directory on WordPress.
2. Activate AI Product Tools from the 'Plugins' menu.
3. Enter and save your OpenAI API key.
4. Customize your API settings as you want.
5. Go to the WooCommerce product and click edit product.
6- Press the "Generate Product Description" button in the product description field and wait.
7- Then save your product.

== Frequently Asked Questions ==

= Is the AI Product Tools plugin free? =
The core features of plugin are 100% free. If you have OpenAI API key, you can use it for free.

= Do I need an API Key? =
Yes, you must have an OpenAI account. If you do not have an account, you must generate one.

= Can I customize Description generation? =
Yes, you can adjust the fields such as OpenAI API Model, Description Language, Description Writing Style, Temperature, Frequency Penalty, Presence Penalty, Top P, Best Of in the management panel of the plugin as you wish.
